# MapMark - Product Overview

## Project Purpose
MapMark is a full-stack social networking application that enables users to connect, share photos, and communicate with friends. The platform focuses on location-based social interactions and photo sharing capabilities.

## Key Features

### User Management
- User registration and authentication with JWT tokens
- Profile management with avatar uploads
- User search functionality with filters (country, city, age range)
- Profile viewing and editing capabilities

### Social Features
- Friend connections and management
- Real-time messaging system
- Photo sharing and gallery management
- Location-based user discovery

### Communication
- Direct messaging between users
- Contact management system
- Friend request handling
- User search and discovery

### Security & Privacy
- CSRF protection for all state-changing operations
- Password hashing with bcrypt
- Rate limiting on authentication endpoints
- Input validation and sanitization
- Privacy policy and terms of service

## Target Users
- Individuals looking to connect with others based on location
- Users who want to share photos and memories
- People seeking new friendships and social connections
- Users interested in location-based social networking

## Use Cases
- **Social Discovery**: Find and connect with people in your area
- **Photo Sharing**: Upload and share photos with friends
- **Messaging**: Communicate directly with connections
- **Profile Management**: Maintain and update personal information
- **Friend Networks**: Build and manage social connections

## Value Proposition
MapMark provides a secure, user-friendly platform for location-based social networking with emphasis on photo sharing and real-time communication, built with modern web technologies and enterprise-grade security practices.
